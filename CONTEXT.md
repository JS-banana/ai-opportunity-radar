# AI&编程活动应用

展示已整理的 AI、编程、黑客马拉松等活动机会，帮助用户发现可参与、可报名、可领取权益或可提交作品的机会。

## Language

**活动机会**:
用户可以报名、提交、领取权益、参与挑战或注册的机会，并且有明确入口、奖励、截止时间或长期状态。
_Avoid_: 活动, 新闻, 资讯, 内容

**目标用户**:
对 AI 相关活动机会感兴趣的人，包括开发者、创作者、学生、产品、运营等，不以开发者为唯一范围。
_Avoid_: 开发者用户, 黑客松选手

**机会发现**:
帮助目标用户快速找到值得参加的活动机会，并判断是否适合自己。
_Avoid_: 活动资讯流, 新闻浏览

**机会详情**:
活动机会的完整决策信息，包括奖励、规则、截止时间、参与方式、官方入口和适合度判断。
_Avoid_: CMS 文章, 活动新闻稿

**应用机会模型**:
网站内部用于展示、筛选和排序活动机会的稳定数据形状，由 Feishu Base 字段映射而来。
_Avoid_: Base 字段, 原始记录

**已接受视觉复刻**:
当前首页和分类页的设计方向，来源于 `docs/design/references/v2-*.png`，实现集中在 `AtlasLanding.tsx` 和 `src/styles/atlas*.css`。
_Avoid_: 临时样式, 占位首页

**即将截止**:
按 `endAt` 派生出的高优先级展示区域，用于首页首屏和滚动后的 Deadline Watch，不是 Feishu 中单独维护的运营栏目。
_Avoid_: 人工置顶列表, CMS 推荐位

**分类浏览**:
一个顶层浏览界面，按用户目标组合机会类型、奖励、地区、形式等派生路径；不是每个标签一页的 SEO 分类站。
_Avoid_: 标签详情页, 独立分类 CMS

**真实数据验收**:
用 live Feishu Base 跑通字段列表、分页记录、mapper、快照体积和跳转入口后，确认 seed 数据可以只作为开发 fallback。
_Avoid_: seed 数据验收, 静态演示数据
