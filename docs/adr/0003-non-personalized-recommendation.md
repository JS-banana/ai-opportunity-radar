# Non-personalized recommendation for MVP

The MVP will rank activity opportunities with deterministic fields such as recommendation score, deadline, reward value, difficulty, official status, and type. Featured or recommended sections are ranking views over the same snapshot, not a manually curated list or separate data layer. The MVP will not include accounts, login, user profiles, favorites, read-later state, deadline reminders, behavior tracking, or AI personalization until real usage proves that these features are necessary.
