# Bilingual UI with source content

The MVP will support Chinese and English routes, UI copy, filters, status labels, enum labels, dates, deadline buckets, and SEO metadata. Locale routes use explicit path prefixes such as `/zh` and `/en`; `/` defaults to `/zh` unless product evidence suggests browser-language negotiation is worth the extra behavior. Long-form activity content such as reward details, participation method, and rules will keep the Feishu Base source text instead of being automatically translated or duplicated in a CMS.
